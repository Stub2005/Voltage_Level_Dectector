# Voltage_Level_Dectector
Analog Electronics Mini Project - Voltage Level Detector Using OpAmps

📌 About the Project
This project is an Analog Electronics mini project that detects different voltage levels using operational amplifiers configured as comparators.

🎯 Objective
To design a circuit that identifies and indicates different voltage ranges using op-amp comparator circuits.

⚙️ Components Used
Operational Amplifier (LM741)
Resistors
LEDs (for indication)
Power Supply

🔌 Working Principle
The op-amp is used as a comparator to compare the input voltage with a reference voltage.
If input voltage > reference voltage → Output goes HIGH
If input voltage < reference voltage → Output goes LOW
Multiple comparators can be used to detect different voltage levels and drive corresponding LEDs.
